DOM HTML (Document Object Model)

DOM HTML (Document Object Model) adalah sebuah model objek yang digunakan untuk merepresentasikan struktur dari sebuah halaman web. Ketika halaman web dimuat dalam peramban (browser), DOM HTML diciptakan sebagai representasi dari elemen-elemen di dalam halaman tersebut dalam bentuk pohon objek. Dengan menggunakan DOM HTML, JavaScript dan bahasa pemrograman lainnya dapat berinteraksi dengan elemen-elemen HTML, mengakses atribut, mengubah konten, serta merespons peristiwa-peristiwa yang terjadi dalam halaman web. DOM HTML memungkinkan pengembang web untuk secara dinamis mengendalikan dan memanipulasi elemen-elemen HTML dalam sebuah halaman.


Dengan model objek ini, JavaScript mendapatkan semua power yang diperlukan untuk menciptakan HTML yang dinamis:

- JavaScript dapat mengubah semua elemen HTML di halaman
- JavaScript dapat mengubah semua atribut HTML di halaman
- JavaScript dapat mengubah semua gaya CSS di halaman
- JavaScript dapat menghapus elemen HTML dan atribut yang ada
- JavaScript dapat menambahkan elemen HTML dan atribut baru
- JavaScript dapat meresponsi semua peristiwa HTML yang ada di halaman
- JavaScript dapat membuat peristiwa HTML baru di halaman


Standar DOM W3C terbagi menjadi 3 bagian yang berbeda:

- model standar untuk semua jenis dokumen
- XML DOM - model standar untuk dokumen XML
- HTML DOM - model standar untuk dokumen HTML
